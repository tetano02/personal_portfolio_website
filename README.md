# Stefano Agnelli | Portfolio

Personal portfolio focused on robotics, AI, and research, with selected projects and contact info.

## Highlights
- Editorial hero with CTAs and social links
- Sections: Projects, About, Education, Experience, Skills, Activities, Contact
- On-scroll animations powered by IntersectionObserver
- Horizontal project scroller with dynamic fades
- Responsive layout with Fraunces + Space Grotesk typography

## Stack
- HTML5
- CSS3 (custom properties, grid/flex layout)
- Vanilla JavaScript

## Structure
- `index.html` markup and content
- `styles.css` theme, layout, animations
- `script.js` animations and scroller behavior
- `assets/` images and CV

## Quick start
1. Open `index.html` in your browser
2. Or run a local server (e.g. Live Server extension)

## Customization
- Edit text and links in `index.html`
- Replace images in `assets/` keeping filenames
- Update colors and typography in `styles.css`
